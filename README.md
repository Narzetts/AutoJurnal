# AutoJurnal

Aplikasi pencatatan jurnal umum otomatis. Anda mengelola transaksi keuangan dengan mudah. Sistem mendeteksi akun debit dan kredit secara otomatis.

## Fitur Utama

*   Input transaksi manual atau batch via Excel.
*   Deteksi akun otomatis berdasarkan kata kunci.
*   Cek keseimbangan neraca secara real-time.
*   Export data kembali ke format Excel.
*   Tampilan responsif untuk desktop dan mobile.
*   Tidak memerlukan server backend.

## Teknologi

*   HTML5
*   CSS3
*   JavaScript Vanilla
*   SheetJS (XLSX)

## Cara Menggunakan

1.  Unduh file index.html.
2.  Buka file tersebut di browser Anda.
3.  Masukkan tanggal, nominal, dan deskripsi transaksi.
4.  Klik tombol Tambah Transaksi.
5.  Sistem mencatat jurnal secara otomatis.

## Format Import Excel

Anda dapat mengimpor data massal. Gunakan kolom berikut.

*   Kolom A, Tanggal (YYYY-MM-DD)
*   Kolom B, Deskripsi Transaksi
*   Kolom C, Nominal (Angka)

## Contoh Kata Kunci

Sistem mengenali kata berikut untuk menentukan akun.

*   Pembelian tunai, Debit Perlengkapan, Kredit Kas
*   Penjualan kredit, Debit Piutang, Kredit Pendapatan
*   Bayar gaji, Debit Beban Gaji, Kredit Kas
